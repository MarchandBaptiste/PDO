# Sakila — Application PHP/PDO

Application web de consultation de la base de données Sakila, réalisée en PHP avec PDO dans le cadre d'un TD à La Manu.

## Pages

- **Accueil** — navigation rapide vers les sections principales
- **Films** — catalogue paginé avec titre, description, durée et note
- **Acteurs** — liste des acteurs avec accès à leur filmographie
- **Magasins** — implantation et informations de contact
- **Données financières** — chiffre d'affaires et stock par boutique
- **Catégories** — liste des genres avec nombre de films
- **Connexion** — authentification avec gestion de session

## Stack

- PHP / PDO
- MySQL (base Sakila)
- HTML / CSS / JavaScript
